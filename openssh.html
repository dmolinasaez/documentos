OpenSSH

Consideraciones

  • Es recomendable realizar copias de seguridad y probar en un entorno de desarrollo antes de aplicar cambios en un entorno productivo
  • Válidar siempre los pasos de ejecución revisando la página web oficial

Chequear versión de SSH

ssh -V
www.openssh.com

Instalar OpenSSH 9.3p2

  1. Descargar OpenSSH 9.3p2 desde la página oficial
  2. Descomprimir el archivo tar.gz 
    tar -xzvf openssh-9.3.tar.gz
  3. Acceder al directorio 
    cd openssh-9.3
  4. Configurar y compilar 
    ./configure make
  5. Instalar 
    make install
  6. Reinicia el servicio SSH 
    restart sshd
